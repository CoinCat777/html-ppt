反例 vs 正例 雙面板
別名：NG vs OK / 反例正例 / 雙面板對比 / 錯誤 vs 正確 / 不夠清楚 vs 這樣說才對

CSS Grid 雙欄。左欄（虛線框 + 淡紅底）= 反例 / 錯誤做法；右欄（實線框 + accent 淡底）= 正確做法。各有標籤徽章、等寬 prompt 框、說明文字或條列。
關鍵：左欄用 border:2px dashed（無填色），右欄用 border:2px solid + accent-subtle 底色，視覺差異清楚。Prompt 框背景用 rgba(255,255,255,0.6) 讓底色透出。

關鍵技巧
需求越細，工具越準
給 Gemini 的 Prompt 品質，決定工具的品質
不夠清楚
「幫我做一個數據分析工具」
AI 不知道資料格式、要做什麼計算、輸出什麼圖——結果幾乎一定要重來
這樣說才對
「輸入欄位 FAB / PRODUCT / LOT，CSV 格式，輸出長條圖，X 軸 LOT，Y 軸 CP 值，顏色按 PRODUCT 分類」
說清楚工具要做什麼事
提供資料格式與欄位名稱
指定輸出型態（圖表類型、軸設定）
