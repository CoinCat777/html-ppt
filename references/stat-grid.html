stat-grid
數字儀表板卡片格
別名：數字儀表板 / numbers dashboard / KPI cards / 數據亮點

CSS Grid 2×3（或 2×2）均分卡片。每張卡片：大數字（accent 色，serif 字型）+ 小標籤（uppercase，muted 色）。卡片背景白色、圓角、淡陰影，整體在暖色背景上清爽突出。
關鍵：數字用大字號（36px+），字型用 serif（Lora / Georgia）強化視覺重量。標籤用 uppercase + letter-spacing 增加精緻感。
