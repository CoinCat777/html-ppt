雙欄垂直時間線
別名：雙欄時間線 / 學習路線 / two-path / 並行路徑 / 雙軌計劃

CSS Grid 雙欄，各欄有標題（accent 底線），每條時間線由 dot（accent 色邊框圓點）+ line（灰色垂直線）+ step-body（標籤 + 說明）組成，最後一步無 line。底部可接全寬洞察橫幅。
關鍵：dot 和 line 在 flex-direction:column 的 step-left 容器內，line 用 flex:1 自動填充步驟間距。最後一步不放 .line 元素。
